Realtime.list_active_storms¶

Realtime.list_active_storms(basin='all')[source]¶

Produces a list of storms currently stored in Realtime.

Parameters

basin (str) – Basin for which to return active storms for. Default is ‘all’.

Returns

list – List containing the storm IDs for currently active storms in the requested basin. Each ID has a Storm object stored as an attribute of Realtime.

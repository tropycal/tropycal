RealtimeStorm.download_graphic_realtime¶

RealtimeStorm.download_graphic_realtime(save_path='')[source]¶

Download the latest official forecast track graphic. Available for both NHC and JTWC sources.

Parameters

save_path (str) – Filepath to save the image in. If blank, default is current working directory.

API Reference¶

tropycal.tracks

Functionality for reading and analyzing storm tracks.

tropycal.tornado

Functionality for reading and analyzing tornado data.

tropycal.rain

Functionality for reading and analyzing tropical cyclone rain data.

tropycal.recon

Functionality for reading and analyzing recon data.

tropycal.realtime

Functionality for managing real-time tropical cyclone data.

tropycal.utils

Collection of basic tools used by Tropycal.

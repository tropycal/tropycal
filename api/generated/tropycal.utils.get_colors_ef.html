get_colors_ef¶

tropycal.utils.get_colors_ef(colormap='default')[source]¶

Retrieve a list of colors for the Enhanced Fujita (EF) tornado scale.

Parameters

colormap (str or list) – Matplotlib colormap to use. Default is ‘default’, which uses Tropycal’s default colors for the EF scale. If a list, this list must have 6 colors in order from EF0 to EF5.

Returns

list or cmap – If used a matplotlib colormap, a cmap is returned, otherwise a list of colors is returned.
